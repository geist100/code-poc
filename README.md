# NextJS micro-frontend example

## How to run:

Run menu-item application
1. `cd menu-item`
2. run `npm install`
3. run `npm run build`
3. run `npm start`

Run main application
1. `cd main`
2. run `npm install`
3. run `npm run build`
4. run `npm start`

Open http://localhost:3000 with your browser to see the result.
